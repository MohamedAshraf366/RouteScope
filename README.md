# RouteScope

Upload a `.kml` or `.kmz` route file to visualize it on a map, estimate drive time by speed, measure area, and convert between KML, KMZ, TAB, and JSON formats.

## Development

You need Node.js and npm installed.

```sh
git clone <this-repository-url>
cd <repository-name>
npm i
npm run dev
```

## Built with

- TanStack Start
- TypeScript
- React
- Tailwind CSS
